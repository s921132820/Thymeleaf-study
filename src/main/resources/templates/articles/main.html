url method 출력내용 html page
/article/list GET 신체 게시글 읽기 성공 list_all.html
/article/{id} GET 7번 게시글 읽기 성공 list_one.html
/articles/create?name=tom&weight=80&height=180 GET 톰의 키는 180, 몸무게는 80 입니다 new.html
/articles/update POST 업데이트 성공 화면 출력 update_ok.html
/articles/{id}/update GET 7번 게시글 수정 폼 로딩 완료 update.html
/articles/{id}/delete GET 3번 글 삭제 완료 delete_ok.html
/articles/{id}/articleComment POST 15번 게시글의 3번째 답글 삭제 완료 delete_ok.html
/articles/{id}/articleComments/{articlecomment-id}/delete POST 11번 게시글의 모든 댓글보기 성공 comment_view.html
